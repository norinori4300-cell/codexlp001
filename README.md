# Pancake Cafe Lumiere

パンケーキ屋さん向けのランディングページです。設計図に沿って、やさしく上品な雰囲気の 1 ページ完結サイトとして作成しています。

## 使用技術

- HTML
- Tailwind CSS（CDN）

## ファイル構成

```text
.
├─ index.html
└─ README.md
```

## 公開方法

`index.html` を GitHub Pages などの静的ホスティングに配置すると、そのまま公開できます。

## 画像の差し替え

画像は `images` フォルダから読み込む構成です。生成用の文面は [IMAGE_PROMPTS.md](./IMAGE_PROMPTS.md) にまとめています。

- 画像保存先: `./images/`
- 画像ファイル名は `IMAGE_PROMPTS.md` の指定名に合わせる
- 画像を置いたあと、そのまま `index.html` で表示されます
