# パンケーキ屋さんLP設計図

## 1. 目的
- パンケーキ屋さんの魅力を、シンプルでやわらかい世界観のランディングページ（LP）で伝える。
- 来店したくなる雰囲気づくりを重視する。
- GitHub で公開しやすいように、**1つの HTML ファイルで完結する構成**を前提にする。
- Codex を使って実装しやすいよう、構成・文言・デザイン方針を明確にする。

## 2. ターゲット
- 主なターゲット：**20代〜60代の女性**
- 想定ニーズ
  - かわいい・落ち着くカフェを探している
  - 写真映えするパンケーキを楽しみたい
  - 友人・家族・一人時間で利用しやすい店を探している
  - メニューや店の雰囲気を事前に見て安心したい

## 3. LPのコンセプト
**「ふわっと甘い、やさしいひととき。」**

### コンセプトの方向性
- シンプル
- ナチュラル
- 上品でやさしい
- 視認性が高い
- 写真が主役

### デザインキーワード
- 余白を広めに取る
- やわらかい配色
- 丸みのあるボタンやカード
- 写真を大きく見せる
- テキスト量は絞り、見やすくする

## 4. 実装前提
- 技術スタック：**HTML + Tailwind CSS**
- **1つの HTML ファイルで完結**
- JavaScript は基本不要
- 画像はダミーとして **Unsplash のプレースホルダー画像**を使用
- GitHub Pages で公開しやすい構成にする

## 5. ページ全体構成
以下の順番で、1ページ完結型のLPにする。

1. ヘッダー
2. ファーストビュー
3. お店の魅力紹介
4. 人気メニュー
5. ドリンクメニュー
6. 店舗の雰囲気・外観
7. お客様が利用しやすいポイント
8. 店舗情報
9. CTA（来店導線）
10. フッター

---

## 6. セクションごとの設計

### 6-1. ヘッダー
#### 役割
- 店名表示
- ナビゲーション表示
- スマホでも見やすい最小構成

#### 掲載内容
- 店名（例：**Pancake Cafe Lumiere**）
- ナビゲーション
  - Home
  - Menu
  - Gallery
  - Access

#### デザイン方針
- 背景は白または淡いクリーム色
- 高さは低めでシンプルにする
- 横幅いっぱいでも、本文領域は中央寄せ

---

### 6-2. ファーストビュー
#### 役割
- 第一印象で「入りたくなる」「食べたくなる」を作る
- 写真を主役にする

#### 掲載内容
- キャッチコピー
- サブコピー
- CTAボタン
- パンケーキの大きなメイン画像

#### 文章案
- キャッチコピー：  
  **ふわっとほどける、しあわせパンケーキ。**
- サブコピー：  
  **やさしい甘さと、くつろぎの空間。毎日頑張るあなたに、ごほうびのカフェ時間を。**
- ボタン文言：  
  - メニューを見る
  - お店を見る

#### ダミー画像例
- メイン画像：  
  `https://source.unsplash.com/featured/1200x900/?pancakes,cafe`

#### デザイン方針
- 左にテキスト、右に画像の2カラム
- スマホでは縦積みにする
- 背景は淡いピンクベージュやアイボリー系
- ボタンは丸みをつける

---

### 6-3. お店の魅力紹介
#### 役割
- この店ならではの良さを短く伝える

#### 掲載内容
3つのポイントをカードで表示

#### 文章案
1. **ふわふわ食感**  
   口の中でほどける、軽やかな焼き上がり。
2. **落ち着く空間**  
   木のぬくもりを感じる、やさしい店内デザイン。
3. **ドリンクとの相性**  
   コーヒーや紅茶と一緒に楽しめる贅沢な時間。

#### デザイン方針
- 3カラムのカード
- アイコンを添えると見やすい
- 背景白、カードに薄い影

---

### 6-4. 人気メニュー
#### 役割
- 来店動機を強めるメインセクション
- 画像で「食べたい」を作る

#### 掲載内容
おすすめパンケーキ3品〜4品

#### メニュー案
1. **ベリーホイップパンケーキ**  
   甘酸っぱいベリーとふんわりホイップの定番人気。
2. **メープルバターパンケーキ**  
   シンプルだからこそ味わえる、王道のおいしさ。
3. **抹茶あずきパンケーキ**  
   和の風味を楽しめる、上品な甘さの一皿。
4. **季節のフルーツパンケーキ**  
   旬のフルーツを彩り豊かに盛り付けた特別メニュー。

#### ダミー画像例
- `https://source.unsplash.com/featured/800x600/?berry,pancakes`
- `https://source.unsplash.com/featured/800x600/?pancake,maple`
- `https://source.unsplash.com/featured/800x600/?matcha,dessert`
- `https://source.unsplash.com/featured/800x600/?fruit,pancakes`

#### デザイン方針
- 2列または4枚カード
- 各カードに画像、商品名、説明、価格欄を配置
- 価格はダミーでよい（例：¥1,200〜¥1,650）

---

### 6-5. ドリンクメニュー
#### 役割
- セット注文をイメージさせる
- カフェとしての魅力を補強する

#### メニュー案
- ブレンドコーヒー
- カフェラテ
- アールグレイティー
- フルーツソーダ
- 自家製レモネード

#### ダミー画像例
- `https://source.unsplash.com/featured/800x600/?coffee,cafe`
- `https://source.unsplash.com/featured/800x600/?tea,cup`
- `https://source.unsplash.com/featured/800x600/?lemonade,drink`

#### デザイン方針

- セット感が出るように余白を整える

### 6-6. 店舗の雰囲気・外観
#### 役割
- お店の安心感と来店イメージを持ってもらう

#### 掲載内容

初めての方でも入りやすく、友人とのカフェ時間にも、一人でのんびり過ごしたい日にもぴったりです。

#### ダミー画像例

- 店内：`https://source.unsplash.com/featured/1200x800/?cafe,interior`

#### デザイン方針
- 大きめ画像を2枚並べる
- スマホでは縦に並べる
- 角丸でやわらかい印象にする

## 7. デザイン仕様

### 7-1. 配色
20代〜60代女性に向けて、甘すぎず上品な色合いにする。

#### 推奨カラー
- メイン背景：`#FFFAF5`
- サブ背景：`#FDF1EC`
- 文字色：`#4B3F3A`
- アクセント：`#E8BFAF`
- ボタン濃色：`#C98F7A`
- 白：`#FFFFFF`

### 7-2. フォントの印象
- 可読性重視
- やわらかい印象
- 見出しは少し華やか
- 本文はシンプルで読みやすくする

### 7-3. 余白
- セクション上下は広め
- カード内も余白をしっかり取る
- 情報を詰め込みすぎない

### 7-4. ボタン
- 角丸大きめ
- ホバーで少し色が濃くなる
- 影は控えめ

### 7-5. カード
- 角丸
- うっすら影
- 画像上、説明下の定番構成

## 8. Tailwind CSS 実装方針

### レイアウト方針
- 最大幅：`max-w-6xl`
- 中央寄せ：`mx-auto`
- 横余白：`px-4 md:px-6 lg:px-8`
- セクション余白：`py-16 md:py-24`

### よく使うクラス例
- 背景：`bg-[#FFFAF5]`
- 見出し：`text-3xl md:text-5xl font-bold`
- 本文：`text-sm md:text-base leading-relaxed`
- カード：`rounded-3xl shadow-sm bg-white`
- ボタン：`inline-block rounded-full px-6 py-3`

### レスポンシブ方針
- スマホファースト
- `md:` 以上で2カラム・3カラム化
- 画像はすべて `object-cover` を基本にする

## 9. Codex に渡す実装指示用メモ
以下の条件で HTML を生成させると実装しやすい。

### 実装条件
- 1ファイル完結の `index.html`
- Tailwind CSS は CDN を使用
- セクションごとにコメントを入れる
- ダミー画像は Unsplash を使用
- 配色はやさしいベージュ・ピンク系
- 写真が主役になるように大きめに配置
- シンプルで上品、女性向け、見やすいUI
- スマホとPCの両方で自然に見えるレスポンシブ対応

### Codex 用指示文の例
```text
パンケーキ屋さんのランディングページを、HTML + Tailwind CSS で1ファイル完結で作成してください。

条件：
- ターゲットは20代〜60代女性
- シンプル、やさしい、上品なデザイン
- セクションは、ヘッダー、ファーストビュー、魅力紹介、人気メニュー、ドリンク、店舗外観・店内、利用しやすいポイント、店舗情報、CTA、フッター
- ダミー画像は Unsplash の URL を使用
- 背景はアイボリーや淡いピンクベージュ
- ボタンやカードは角丸でやわらかい印象
- レスポンシブ対応
- 外部CSSファイルは使わず、Tailwind CDN で完結
- コードは読みやすく、セクションごとにコメントをつける
```

## 11. 追加アイデア
シンプルなHPに少しだけ魅力を足すなら、次の要素が相性がよい。

### アイデア1：季節限定メニューの小バナー
- 春限定いちご
- 秋限定モンブラン
- 期間限定感で興味を引ける

### アイデア2：小さな口コミ風テキスト
- 「ふわふわで軽くて食べやすい」
- 「店内が落ち着いていて過ごしやすい」
- LPに親しみやすさが出る

### アイデア3：やさしい装飾
- 背景にうっすら丸やぼかし
- セクションの切り替わりに波線風の余白
- ただし装飾しすぎない

### アイデア4：インスタ風ギャラリー
- 3〜4枚の写真を並べる
- 写真メインで雰囲気訴求できる

## 12. このLPで大事にすること
- 情報を詰め込みすぎない
- 写真を大きく見せる
- 甘すぎず上品にまとめる
- 初めて見た人が、迷わず雰囲気とメニューを理解できるようにする
- GitHub でそのまま公開しやすいシンプルな実装にする

---

## 13. 仕上がりイメージ
このLPは、**「写真が主役の、やさしくて見やすいカフェLP」**を目指す。

特に重要なのは以下の3点。
- ファーストビューで食べたくなる
- メニュー画像で魅力が伝わる
- 店舗外観で安心感が出る

この3点を押さえることで、シンプルでも十分に魅力的なページになる。
